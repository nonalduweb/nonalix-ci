# Règles de comportement de l'Agent

## Déploiement et Push Git
- **NE JAMAIS** déployer le site, pousser des modifications sur la branche de production (`main`), ou lancer des scripts de déploiement sans l'autorisation explicite préalable de l'utilisateur.
- Toujours demander l'avis de l'utilisateur avant d'exécuter `git push` ou de modifier les pipelines de CI/CD.
